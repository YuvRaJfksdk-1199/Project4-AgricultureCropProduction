# Notebooks

## agriculture_analysis.ipynb

Complete Jupyter notebook for the Agriculture Crop Production Prediction project.

### How to use:

**Option 1: Google Colab (Recommended)**
```
1. Open colab.research.google.com
2. Click "Upload" → select this notebook
3. Download dataset from Google Drive link in README
4. Upload to Colab when prompted
5. Run all cells
```

**Option 2: Local Jupyter**
```bash
pip install jupyter
jupyter notebook agriculture_analysis.ipynb
```
